# シューズの設計ファイル(stl)

- pushPrt
  - 内部の押し出し機構の細かい部品
  - "Base"と付いている部品をもとに組み立て
- shell
  - 外側
  - 実物だと白くて柔らかい素材(TPU)でプリントしてあるところ
- table.stl
  - どこ点シューズ専用点字ブロック
